
# 🦁 Московский зоопарк: Тотемное животное — Telegram-бот

Telegram-бот с викториной, который помогает пользователю узнать своё «тотемное животное» среди обитателей Московского зоопарка и знакомит с программой опеки «Клуб друзей».

Проект выполнен в рамках учебного курса по Python-разработке.

## 📌 О проекте

Московский зоопарк стремится привлечь внимание взрослых людей к программе опеки животных. Бот решает две задачи:
- Развлекательная: пользователь проходит викторину из 7 вопросов и получает персональный результат.
- Информационная: бот рассказывает о программе опеки и помогает связаться с сотрудниками зоопарка.

Все вопросы и животные основаны на реальных данных с сайта Московского зоопарка. Итоговое «тотемное животное» выбирается по накопительной балльной системе (при равном количестве баллов — случайный выбор из лидеров).

## ✨ Возможности бота

- 🚀 Простая команда `/start` и кнопка «Начать викторину»
- 🧩 7 вопросов с 6 вариантами ответов 
- 📊 Алгоритм подсчёта баллов и определение тотемного животного
- 🖼 Вывод реалистичного изображения животного с описанием
- ℹ️ Кнопка «Узнать о программе опеки» с общей информацией
- 📤 Возможность поделиться результатом в Telegram (inline-режим + ссылка на бота)
- 📞 Связь с сотрудником зоопарка: результат пересылается администратору, пользователю показывается почта
- 📝 Сбор отзывов (сохраняются в файл)
- 🔄 Повторное прохождение викторины
- 🛡 Защита от повторных нажатий на старые кнопки

## 🚀 Быстрый запуск (локально)

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/lavren007/moscow_zoo_bot.git
cd moscow_zoo_bot
```

### 2. Создайте и активируйте виртуальное окружение (рекомендуется)
```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# macOS / Linux:
source venv/bin/activate
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Настройте конфигурацию
Откройте файл `config.py` и укажите **токен вашего бота** и **ID администратора** (сотрудника, который будет получать заявки).

> ⚠️ **Важно!** Не публикуйте реальный токен в открытом доступе. Для продакшена используйте переменные окружения.
А в коде уже подставьте значение из переменной окружения.

Для теста сейчас достаточно заменить строки в `config.py`:
```python
BOT_TOKEN = "ваш_токен_от_BotFather"
ADMIN_CHAT_ID = 123456789  # id сотрудника в Telegram
```

Как узнать ID пользователя:
- Попросите сотрудника написать боту [@userinfobot](https://t.me/userinfobot)

### 5. Добавьте изображения животных (если они не добавлены)
В папку `images/` положите 6 файлов с именами:
- `tiger.jpg`
- `polar_bear.jpg`
- `meerkat.jpg`
- `flamingo.jpg`
- `elephant.jpg`
- `snow_leopard.jpg`

Размеры — на ваше усмотрение, бот отправляет их как есть. Если какого-то изображения нет, бот выведет текст без картинки.

### 6. Запуск
```bash
python bot.py
```

Если в консоли появилось `INFO:aiogram.dispatcher:Start polling` — бот работает. Найдите его в Telegram по юзернейму и нажмите **/start**.

## 🧪 Как работает алгоритм

- Каждый ответ добавляет определённому животному  2 балла.
- После 7 вопросов побеждает животное с наибольшей суммой баллов.
- При равенстве баллов у нескольких животных — победитель выбирается случайным образом (`random.choice(leaders)`).
- Результат запоминается, чтобы при нажатии кнопок «Об опеке» или «Связаться» использовалось именно оно.
