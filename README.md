
# 🐾 Зоопарк: Найди своего покровителя

Telegram-бот с викториной, который подбирает пользователю тотемное животное среди обитателей Московского зоопарка и рассказывает о программе опеки.

## 🎯 О чём проект

Бот помогает взрослым посетителям зоопарка узнать о программе опеки через игру:
- Проходит викторину из 7 вопросов
- Получает своё тотемное животное
- Узнаёт, как стать опекуном

Результат выбирается по системе баллов (при равном счёте — случайный выбор).

## ✨ Что умеет бот

- `/start` и кнопка запуска
- 7 вопросов, 6 вариантов ответов
- Подсчёт баллов и определение тотема
- Картинка + описание животного
- Информация об опеке
- Поделиться результатом в Telegram
- Запрос на связь с сотрудником
- Отзывы (сохраняются в файл)
- Повторное прохождение

## ⚙️ Как запустить

### 1. Установка проекта

```bash
git clone https://github.com/
cd moscow_zoo_bot


### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройте конфигурацию
Откройте файл `config.py` и укажите **токен вашего бота** и **ID администратора** (сотрудника, который будет получать заявки).

### 5. Запуск
```bash

```
